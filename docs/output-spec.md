# 生成物仕様書（出力フォーマット）

AI支援会社向け「企業別AI活用レポート」の**出力**の型を定義する。  
クオリティを維持するため、この仕様に沿った列・行構造でスプレッドシート（CSV / Google Sheets）を出力する。

---

## 1. 出力の目的

- 顧客企業ごとに「AIの活用方法」と「期待効果」を一覧化し、提案・商談・レポートにそのまま使える形で渡す。
- スプレッドシートで管理・フィルタ・共有しやすくする。

---

## 2. シート構成（2シート想定）

### シート1: 企業サマリ（1企業1行）

| 列名 | 説明 | 例 |
|------|------|-----|
| 企業名 | 正式名称 | 株式会社サンプル |
| 業種 | 大分類 | 製造業 / IT・ SaaS / 小売 など |
| 事業概要 | 1〜2文 | 〇〇の開発・販売を行っている |
| 公式URL | ホームページURL | https://example.co.jp |
| 取得日時 | レポート生成日時（ISO8601推奨） | 2026-02-20T12:00:00+09:00 |
| 活用領域メモ | 入力で指定した注目領域（あれば） | 営業・開発・カスタマーサポート |

### シート2: 活用案・効果一覧（1案1行、企業ごとに複数行）

| 列名 | 説明 | 例 |
|------|------|-----|
| 企業名 | 上記と同一（紐付け用） | 株式会社サンプル |
|  No | 同一企業内の連番 | 1, 2, 3 … |
| 活用シーン | どこで使うか（部門・業務） | 営業：提案書・見積もり作成 |
| 活用方法 | 具体的に何をどうするか（1〜3文） | 過去案件を学習させ、テンプレ＋差分入力で提案書を自動生成。見積もりは単価マスタと連携して算出。 |
| 期待効果（定性的） | 業務改善・品質・速度など | 提案書作成時間を約50%削減、表現のばらつきを抑制 |
| 期待効果（定量的） | 数値目標があれば（不明は空欄可） | 工数削減 月20h想定 |
| 優先度 | 導入しやすさ・インパクトの目安 | 高 / 中 / 低 |
| 備考 | 前提条件・注意点・参照元 | 既存CRMとの連携が必要 |

---

## 3. CSV 出力時のルール

- 文字コード: UTF-8（BOM付き推奨、Excelで開く場合）
- 区切り: カンマ `,`
- 改行・カンマ・ダブルクォートを含むセルは `"..."` で囲み、内部の `"` は `""` にエスケープ
- シート1・シート2は別ファイルでもよい: `{企業ID}_summary.csv`, `{企業ID}_utilization.csv`  
  または1ファイルに「サマリ行＋空行＋活用案行」をまとめても可（運用で選択）

---

## 4. クオリティ基準（生成時に守ること）

- **活用方法**: その企業の事業・業種に紐づいた具体案にする（汎用フレーズの羅列にしない）。
- **期待効果**: 可能な範囲で「誰の・何が・どう変わるか」を一文で書く。数値は根拠が分かる場合のみ記載し、推測の場合は「想定」「目安」と明記。
- **禁止**: 他社の機密・固有名詞の誤用、投資助言・医療診断に見える表現。

---

## 5. 既存の「生成物」仕様との差し替え

すでに「こういう表で出したい」という仕様（Excel/Notionなど）がある場合は、  
この `output-spec.md` をその内容に差し替え、列名・必須項目だけ揃えてもらえれば、  
同じ入力仕様・エージェント設計のまま出力フォーマットだけ差し替えて使える。
